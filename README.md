# 📖 Projeto Site de Receitas - Rocketseat

Projeto 1 da formação Full Stack da Rocketseat, desenvolvido durante o módulo "Iniciando no HTML e CSS". O objetivo foi criar uma página web estática para exibir uma receita, praticando a estruturação de conteúdo com HTML semântico e a estilização básica com CSS.

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3

## ✨ Aprendizados Principais

- Estruturação semântica do HTML (`<header>`, `<section>`, `<footer>`, etc.).
- Estilização básica com CSS:
    - Seletores CSS (tags, IDs).
    - Propriedades de estilo (cores, fontes, background, espaçamento, sombras).
    - Box Model (margin, padding, border).
    - Importação de fontes externas (Google Fonts).
    - Uso de imagens de fundo e elementos.
